# Number Guessing Game

A simple command-line number guessing game built with Python.

## Overview

This game prompts the player to guess a randomly generated number within a specified range. The player receives feedback on whether their guess is too high, too low, or correct.

## Features

- Random number generation
- User input validation
- Attempt counter
- Feedback messages

## Installation

1. Clone or download the repository
2. Ensure Python 3.x is installed

## Usage

Run the game with:

```bash
python main.py
```

Follow the on-screen prompts to guess the number.

## How to Play

1. The game generates a random number
2. Enter your guess when prompted
3. Receive feedback if your guess is too high or too low
4. Continue until you guess correctly
5. View your total attempts

## Requirements

- Python 3.6+
